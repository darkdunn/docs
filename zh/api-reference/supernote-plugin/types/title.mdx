---
title: "Title"
description: "笔记中的标题数据结构。"
---

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `X` | `number` | 标题左上角坐标 X（像素点坐标）|
| `Y` | `number` | 标题左上角坐标 Y（像素点坐标） |
| `width` | `number` | 标题宽度（像素） |
| `height` | `number` | 标题高度（像素） |
| `page` | `number` | 标题所在页码 |
| `num` | `number` | 标题在所在页的序号 |
| `style` | `number` | 标题样式：`0` 去掉标题属性，`1` 黑底，`2` 灰白，`3` 灰黑，`4` 阴影 |
| `controlTrailNums` | `number[]` | 关联的笔划编号列表，表示哪些笔画是属于该标题的 |
