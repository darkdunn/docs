---
title: "setSlideBarStatus"
description: "设置滑条是否可滑动。"
---

```ts
static setSlideBarStatus(status: boolean): Promise<boolean>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `status` | `boolean` | `true` 可滑动，`false` 不可滑动 |

## 示例

```ts wrap
import { PluginCommAPI } from 'rtn-supernote-plugin-core';

/**
 * 设置滑条可用状态的示例。
 */
export async function exampleSetSlideBarStatus() {
  const ok = await PluginCommAPI.setSlideBarStatus(false);
  return ok;
}
```
