---
title: "setStatusBarAndSlideBarState"
description: "设置下拉菜单与滑条的锁定状态。"
---

```ts
static setStatusBarAndSlideBarState(isLock: boolean): void;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `isLock` | `boolean` | `true` 锁住不可用，`false` 解锁可用 |

## 示例

```ts wrap
import { PluginCommAPI } from 'rtn-supernote-plugin-core';

/**
 * 锁定/解锁下拉菜单与滑条的示例。
 */
export async function exampleSetStatusBarAndSlideBarState() {
  PluginCommAPI.setStatusBarAndSlideBarState(true);
}
```
