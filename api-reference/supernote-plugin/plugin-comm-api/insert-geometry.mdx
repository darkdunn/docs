---
title: "insertGeometry"
description: "插入几何图形。"
---

```ts
static insertGeometry(geometry: Geometry): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `geometry` | [`Geometry`](/api-reference/supernote-plugin/types/geometry) | 几何图形参数 |

## 示例

```ts wrap
import { PluginCommAPI, type Geometry } from 'rtn-supernote-plugin-core';

/**
 * 插入一个简单几何图形的示例。
 */
export async function exampleInsertGeometry() {
  const geometry: Geometry = {
    penColor: 0x9d,
    penType: 10,
    penWidth: 2,
    type: 'GEO_polygon',
    points: [
      { x: 100, y: 100 },
      { x: 200, y: 100 },
      { x: 200, y: 200 },
      { x: 100, y: 200 },
      { x: 100, y: 100 },
    ],
    ellipseCenterPoint: null,
    ellipseMajorAxisRadius: 0,
    ellipseMinorAxisRadius: 0,
    ellipseAngle: 0,
  };

  const res = await PluginCommAPI.insertGeometry(geometry);
  if (!res.success) {
    throw new Error(res.error?.message ?? 'insertGeometry 调用失败');
  }
  return res.result;
}
```
