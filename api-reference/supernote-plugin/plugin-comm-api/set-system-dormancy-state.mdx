---
title: "setSystemDormancyState"
description: "设置系统休眠状态。"
---

```ts
static setSystemDormancyState(enable: boolean): void;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `enable` | `boolean` | `true` 阻止系统休眠，`false` 开启休眠 |

## 示例

```ts wrap
import { PluginCommAPI } from 'rtn-supernote-plugin-core';

/**
 * 控制系统休眠的示例。
 */
export async function exampleSetSystemDormancyState() {
  PluginCommAPI.setSystemDormancyState(true);
}
```
