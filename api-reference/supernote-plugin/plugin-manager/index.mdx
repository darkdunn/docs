---
title: "PluginManager"
description: "插件生命周期、按钮与事件监听的管理入口。"
---

`PluginManager` 是插件运行的管理入口，负责：

- 初始化插件环境
- 监听插件生命周期
- 注册/监听插件按钮与配置按钮
- 注册/监听插件事件
- 获取插件基础信息（名称、目录、设备类型等）

```ts
import { PluginManager } from 'rtn-supernote-plugin-core';
```
