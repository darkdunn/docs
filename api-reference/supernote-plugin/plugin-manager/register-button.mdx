---
title: "registerButton"
description: "注册插件按钮。"
---

```ts
registerButton(type: number, appTypes: string[], button: Object): Promise<boolean>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `type` | `number` | 按钮类型：`1` 侧边栏，`2` 套索工具栏，`3` 文档划词工具条 |
| `appTypes` | `string[]` | 应用类型数组：`NOTE`/`DOC`（会自动归一化与去重） |
| `button` | `PluginButton` | 按钮信息对象 |

**返回**

- `Promise<boolean>`：是否注册成功
