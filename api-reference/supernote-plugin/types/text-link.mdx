---
title: "TextLink"
description: "Text Link Class"
---


| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `destPath` | `string` | Link Target Path. If the link type is 4 (url), fill in the url address here. |
| `destPage` | `number` | Target Page Number |
| `style` | `number` | Link style: 0: solid underline, 1: solid border, 2: dashed border |
| `linkType` | `number` | Link types: 0: Jump to note page, 1: Jump to note file, 2: Document, 3: Picture, 4: Website, 6: Excerpt link |
| `rect` | [`rect`](/api-reference/supernote-plugin/types/rect) | Link Display Region |
| `fontSize` | `number` | Font Size |
| `showText` | `string` | The text content displayed in the note is an abbreviated version of the fullText. |
| `fullText` | `string` | The link is the text content |
| `isItalic` | `number` | Italic: 0: italic, 1: not italic |
