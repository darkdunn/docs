---
title: "ButtonListener"
description: "插件按钮点击事件监听器接口。"
---

```ts
interface ButtonListener {
  onButtonPress(event: { id: number; name: string; icon: string }): void;
}
```

**说明**

- 通过 [`registerButtonListener`](/api-reference/supernote-plugin/plugin-manager/register-button-listener) 注册监听
- `event` 字段含义参考：[插件界面 - 按钮事件监听](/plugin-base/plugin-ui)

## 示例

```ts wrap
import { PluginManager } from 'rtn-supernote-plugin-core';

export async function exampleButtonListener() {
  const sub = PluginManager.registerButtonListener({
    onButtonPress(event) {
      console.log(event.id);
    },
  } as any);
  return sub;
}
```
