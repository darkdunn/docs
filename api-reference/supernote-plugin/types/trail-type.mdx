---
title: "ElementType"
description: "用于标识 Element 的类型（常量定义）。"
---

| 常量 | 值 | 说明 |
| --- | ---:| --- |
| `ElementType.TYPE_STROKE` | `0` | 笔划（手写笔迹） |
| `ElementType.TYPE_TITLE` | `100` | 标题 |
| `ElementType.TYPE_TEXT` | `500` | 普通文本框 |
| `ElementType.TYPE_TEXT_DIGEST_QUOTE` | `501` | 引用摘录文本框 |
| `ElementType.TYPE_TEXT_DIGEST_CREATE` | `502` | 创建摘录文本框 |
| `ElementType.TYPE_LINK` | `600` | 链接 |
| `ElementType.TYPE_GEO` | `700` | 几何图形 |
