---
title: "ButtonSubscription"
description: "按钮监听订阅对象。"
---

```ts
type ButtonSubscription = {
  remove(): void;
};
```

**说明**

- 由 [`registerButtonListener`](/api-reference/supernote-plugin/plugin-manager/register-button-listener) 返回
- 调用 `remove()` 注销按钮事件监听

## 示例

```ts wrap
import { PluginManager } from 'rtn-supernote-plugin-core';

export async function exampleButtonSubscription() {
  const sub = PluginManager.registerButtonListener({
    onButtonPress() {},
  } as any);
  sub.remove();
}
```
