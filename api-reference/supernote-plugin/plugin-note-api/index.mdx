## 导入

```ts
import { PluginNoteAPI } from "rtn-supernote-plugin-core";
```

## 返回值约定

多数异步接口返回 [`APIResponse<T>`](/api-reference/supernote-plugin/types/api-response)。通常只在 `success === true` 时读取 `result`。
