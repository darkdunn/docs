---
title: "getLassoLinks"
description: "获取套索链接数据。"
---
<Note>
必须套索之后调用该接口，返回会调用失败。

建议在点击套索工具条按钮后进入的插件界面调用该接口。
</Note>

```ts
static getLassoLink(): Promise<APIResponse<LassoLink[]>>;
```

**返回**

- [`APIResponse<LassoLink[]>`](/api-reference/supernote-plugin/types/api-response)：
`result` 为[`LassoLink`](/api-reference/supernote-plugin/types/lasso-link)对象数组。

## 示例

```ts wrap
import { PluginNoteAPI } from 'rtn-supernote-plugin-core';

/**
 * 获取当前套索的链接数据示例。
 */
export async function exampleGetLassoLink() {
  const res = await PluginNoteAPI.getLassoLink();
  if (!res.success) {
    throw new Error(res.error?.message ?? 'getLassoLink 调用失败');
  }
  return res.result;
}
```
