---
title: "getLassoTitle"
description: "获取套索的标题。"
---

<Note>
必须套索之后调用该接口，返回会调用失败。

建议在点击套索工具条按钮后进入的插件界面调用该接口。
</Note>

```ts
static getLassoTitle(): Promise<APIResponse<Title[]>>;
```

**返回**

- [`APIResponse<Title[]>`](/api-reference/supernote-plugin/types/api-response)：
`result` 为[`Title`](/api-reference/supernote-plugin/types/title)对象数组。
