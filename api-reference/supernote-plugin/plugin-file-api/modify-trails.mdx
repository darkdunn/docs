---
title: "modifyElements"
description: "修改已存在的页面元素数据（必须是已存在的数据，否则不会生效）。"
---

```ts
static modifyElements(notePath: string, page: number, elements: Element[]): Promise<APIResponse<number[]>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页面索引（从 1 开始） |
| `elements` | [`Element[]`](/api-reference/supernote-plugin/types/trail) | 需要修改的元素数组（必须已存在） |

**返回**

- `APIResponse<number[]>`：`result` 为修改成功的元素索引列表
