---
title: "replaceElements"
description: "替换页面元素数据。"
---

```ts
static replaceElements(notePath: string, page: number, elements: Element[]): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页面索引（从 1 开始） |
| `elements` | [`Element[]`](/api-reference/supernote-plugin/types/trail) | 元素数据数组 |

**返回**

- `APIResponse<boolean>`：`result === true` 表示替换成功
