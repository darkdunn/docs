---
title: "clearPageLayerData"
description: "清除页面指定图层数据（仅可操作笔记文件）。"
---

```ts
static clearPageLayerData(notePath: string, page: number, layer: number): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页码（从 1 开始） |
| `layer` | `number` | 图层号 |

**返回**

- `APIResponse<boolean>`：`result === true` 表示清除成功
