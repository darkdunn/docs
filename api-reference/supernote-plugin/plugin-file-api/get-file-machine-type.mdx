---
title: "getFileMachineType"
description: "获取创建该笔记或旁注文件的机器类型。"
---

```ts
static getFileMachineType(notePath: string): Promise<APIResponse<number>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 文件路径 |

**返回**

- `APIResponse<number>`：`result` 为机器型号：`0` A5，`1` A6，`2` A6X，`3` A5X，`4` nomad，`5` Manta

## 示例

```ts wrap
import { PluginFileAPI } from 'rtn-supernote-plugin-core';

/**
 * 获取文件创建机型的示例。
 */
export async function exampleGetFileMachineType() {
  const notePath = '/storage/emulated/0/Note/demo.note';
  const res = await PluginFileAPI.getFileMachineType(notePath);
  if (!res.success) {
    throw new Error(res.error?.message ?? 'getFileMachineType 调用失败');
  }
  return res.result;
}
```
