---
title: "sortLayers"
description: "对图层信息排序（最前面的图层在最顶端）。"
---

```ts
static sortLayers(notePath: string, page: number, layerIds: number[]): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页面索引（从 1 开始） |
| `layerIds` | `number[]` | 需要排序的图层 ID 列表（顺序即层级顺序） |

**返回**

- `APIResponse<boolean>`：`result === true` 表示排序成功

## 示例

```ts wrap
import { PluginFileAPI } from 'rtn-supernote-plugin-core';

/**
 * 排序图层的示例（传入期望的层级顺序）。
 */
export async function exampleSortLayers() {
  const notePath = '/storage/emulated/0/Note/demo.note';
  const page = 1;
  const res = await PluginFileAPI.sortLayers(notePath, page, [102, 101, 100]);
  if (!res.success) {
    throw new Error(res.error?.message ?? 'sortLayers 调用失败');
  }
  return res.result;
}
```
