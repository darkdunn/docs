---
title: "insertLayer"
description: "插入图层数据。"
---

```ts
static insertLayer(notePath: string, page: number, layer: Layer): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页面索引（从 1 开始） |
| `layer` | `Layer` | 需要插入的图层数据 |

**返回**

- `APIResponse<boolean>`：`result === true` 表示插入成功

## 示例

```ts wrap
import { PluginFileAPI } from 'rtn-supernote-plugin-core';

/**
 * 插入图层的示例。
 */
export async function exampleInsertLayer() {
  const notePath = '/storage/emulated/0/Note/demo.note';
  const page = 1;
  const layer = { id: 100, name: '新图层', enable: true } as any;
  const res = await PluginFileAPI.insertLayer(notePath, page, layer);
  if (!res.success) {
    throw new Error(res.error?.message ?? 'insertLayer 调用失败');
  }
  return res.result;
}
```
