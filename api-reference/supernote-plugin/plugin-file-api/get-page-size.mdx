---
title: "getPageSize"
description: "获取指定页面的尺寸（像素）。"
---

```ts
static getPageSize(notePath: string, page: number): Promise<APIResponse<Size>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页码（从 1 开始） |

**返回**

- `APIResponse<Size>`：`result` 为页面尺寸，见 [`Size`](/api-reference/supernote-plugin/types/size)
