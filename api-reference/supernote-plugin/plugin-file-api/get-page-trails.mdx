---
title: "getElements"
description: "获取页面元素数据，包含普通笔划、链接、标题、文本框。"
---

```ts
static getElements(page: number, notePath: string): Promise<APIResponse<Element[]>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `page` | `number` | 页面索引（从 1 开始） |
| `notePath` | `string` | 笔记路径 |

**返回**

- `APIResponse<Element[]>`：元素类型见 [`Element`](/api-reference/supernote-plugin/types/trail)
