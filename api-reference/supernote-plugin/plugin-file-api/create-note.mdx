---
title: "createNote"
description: "创建笔记。"
---

```ts
static createNote(params: {
  notePath: string;
  template: string;
  mode: number;
  isPortrait: boolean;
}): Promise<APIResponse<boolean>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `params.notePath` | `string` | 笔记路径 |
| `params.template` | `string` | 模版路径 |
| `params.mode` | `number` | 创建模式：`0` 普通模式，`1` 识别布局 |
| `params.isPortrait` | `boolean` | 是否竖屏 |

**返回**

- `APIResponse<boolean>`：`result === true` 表示创建成功

## 示例

```ts wrap
import { PluginFileAPI } from 'rtn-supernote-plugin-core';

/**
 * 创建笔记的示例。
 */
export async function exampleCreateNote() {
  const res = await PluginFileAPI.createNote({
    notePath: '/storage/emulated/0/Note/new.note',
    template: '/storage/emulated/0/Note/template.note',
    mode: 0,
    isPortrait: true,
  });
  if (!res.success) {
    throw new Error(res.error?.message ?? 'createNote 调用失败');
  }
  return res.result;
}
```
