---
title: "getPageRotationType"
description: "获取指定页面的旋转角度类型。"
---

```ts
static getPageRotationType(notePath: string, page: number): Promise<APIResponse<number>>;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `notePath` | `string` | 笔记路径 |
| `page` | `number` | 页码（从 1 开始） |

**返回**

- `APIResponse<number>`：`result` 为旋转角度类型（文档约定：`1000`/`2000`/`1090`/`2090`/`1180`/`2180`/`1270`/`2270`）

## 示例

```ts wrap
import { PluginFileAPI } from 'rtn-supernote-plugin-core';

/**
 * 获取页面旋转类型的示例。
 */
export async function exampleGetPageRotationType() {
  const notePath = '/storage/emulated/0/Note/demo.note';
  const page = 1;
  const res = await PluginFileAPI.getPageRotationType(notePath, page);
  if (!res.success) {
    throw new Error(res.error?.message ?? 'getPageRotationType 调用失败');
  }
  return res.result;
}
```
