---
title: "emrPoint2Android"
description: "将 EMR 坐标点转换为屏幕像素点。"
---

```ts
static emrPoint2Android(point: Point, pageSize: { width: number; height: number }): Point;
```

**参数**

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `point` | [`Point`](/api-reference/supernote-plugin/types/point) | EMR 坐标点 |
| `pageSize` | [`Size`](/api-reference/supernote-plugin/types/size) | 页面像素尺寸 |

**返回**

- [`Point`](/api-reference/supernote-plugin/types/point)：转换后的屏幕像素点
