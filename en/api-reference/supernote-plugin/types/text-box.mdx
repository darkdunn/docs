---
title: "TextBox"
description: "Text box data in notes."
---

| Field | Type | Description |
| --- | --- | --- |
| `fontSize` | `number` | Font size |
| `fontPath` | `string \| null` | Font path |
| `textContentFull` | `string \| null` | Full text content |
| `textRect` | [`Rect`](/api-reference/supernote-plugin/types/rect) | Text rectangle (pixel units) |
| `textDigestData` | `string \| null` | Digest data (exists for digest text boxes; cannot be inserted) |
| `textAlign` | `number` | Alignment: `0` left, `1` center, `2` right |
| `textBold` | `number` | Bold: `0` normal, `1` bold |
| `textItalics` | `number` | Italic: `0` normal, `1` italic |
| `textFrameWidthType` | `number` | Frame width type: `0` fixed width, `1` adaptive width |
| `textFrameStyle` | `number` | Frame style: `0` none, `3` stroke |
| `textEditable` | `number` | Editability: `0` editable, `1` not editable |
