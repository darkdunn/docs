---
title: "LassoLink"
description: "Link data for lasso-selected strokes."
---

| Field | Type | Description |
| --- | --- | --- |
| `category` | `number` | `1` for stroke link |
| `controlTrailNums` | `number[]` | Stroke indices |
| `linkType` | `number` | Type: `0` note page, `1` note file, `2` document, `3` image, `4` url, `5` other, `6` digest |
| `destPath` | `string` | Target path |
| `destPage` | `number` | Target page |
