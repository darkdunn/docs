---
title: "PenInfo"
description: "Pen information."
---

| Field | Type | Description |
| --- | --- | --- |
| `type` | `number` | Pen type: `10` fineliner, `1` pressure pen, `11` marker, `14` calligraphy |
| `color` | `number` | Pen color value: `0x00` black, `0x9D` dark gray, `0xC9` light gray, `0xFE` white |
| `width` | `number` | Pen width (minimum `100`) |
