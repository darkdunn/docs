---
title: "Title"
description: "Title data in notes."
---

| Field | Type | Description |
| --- | --- | --- |
| `X` | `number` | Top-left X (pixel coordinates) |
| `Y` | `number` | Top-left Y (pixel coordinates) |
| `width` | `number` | Width (px) |
| `height` | `number` | Height (px) |
| `page` | `number` | Page number |
| `num` | `number` | Order number within the page |
| `style` | `number` | Style: `0` remove title attr, `1` black background, `2` gray-white, `3` gray-black, `4` shadow |
| `controlTrailNums` | `number[]` | Stroke indices associated with this title |
