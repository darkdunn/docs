---
title: "PointUtils"
description: "Convert between EMR (digitizer) and pixel coordinate systems."
---

`PointUtils` helps convert coordinates between:

- Pixel coordinates: used by screen rendering/layout (px)
- EMR coordinates: used by the digitizer hardware (hardware ticks)

## Constants

### Orientation

| Const | Value | Description |
| --- | ---:| --- |
| `PointUtils.ROTATION_0` | `1000` | 0° portrait |
| `PointUtils.ROTATION_0_LR` | `2000` | 0° portrait, left-right pagination |
| `PointUtils.ROTATION_90` | `1090` | 90° landscape |
| `PointUtils.ROTATION_90_UD` | `2090` | 90° landscape, up-down pagination |
| `PointUtils.ROTATION_180` | `1180` | 180° portrait |
| `PointUtils.ROTATION_180_LR` | `2180` | 180° portrait, left-right pagination |
| `PointUtils.ROTATION_270` | `1270` | 270° landscape |
| `PointUtils.ROTATION_270_UD` | `2270` | 270° landscape, up-down pagination |

### Machine Type

| Const | Value | Description |
| --- | ---:| --- |
| `PointUtils.MACHINE_TYPE_A5` | `0` | A5 |
| `PointUtils.MACHINE_TYPE_A6` | `1` | A6 |
| `PointUtils.MACHINE_TYPE_A6X` | `2` | A6X |
| `PointUtils.MACHINE_TYPE_A5X` | `3` | A5X |
| `PointUtils.MACHINE_TYPE_NOMAD` | `4` | Nomad |
| `PointUtils.MACHINE_TYPE_MANTA` | `5` | Manta |

### Common Page Sizes

| Const | Value |
| --- | --- |
| `PointUtils.NORMAL_PAGE_SIZE` | `{ width: 1404, height: 1872 }` |
| `PointUtils.A5X2_PAGE_SIZE` | `{ width: 1920, height: 2560 }` |
