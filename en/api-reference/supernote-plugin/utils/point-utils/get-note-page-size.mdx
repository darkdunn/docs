```ts
static getNotePageSize(orientation: number, machineType: number): { width: number; height: number };
```

**Parameters**

| Parameter | Type | Description |
| --- | --- | --- |
| `orientation` | `number` | page orientation. See `PointUtils.ROTATION_*` |     
| `machineType` | `number` | device model type. See `PointUtils.MACHINE_TYPE_*` |

**Returns**

- [`Size`](/api-reference/supernote-plugin/types/size):pagepixelsize

## Example

```ts wrap
import { PointUtils } from 'sn-plugin-lib';

/**
 * derivepagepixelsize example.
 */
export async function exampleGetNotePageSize() {
 return PointUtils.getNotePageSize(PointUtils.ROTATION_0, PointUtils.MACHINE_TYPE_A5X);
}
```
