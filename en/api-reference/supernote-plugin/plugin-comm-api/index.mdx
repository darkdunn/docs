## Import

```ts wrap
import { PluginCommAPI } from "rtn-supernote-plugin-core";
```

## Return Convention

Most async APIs return [`APIResponse<T>`](/api-reference/supernote-plugin/types/api-response). Read `result` when `success === true`. If `success === false`, `result` is empty and `error` contains details.
