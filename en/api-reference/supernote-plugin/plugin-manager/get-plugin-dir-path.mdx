---
title: "getPluginDirPath"
description: "Get plugininstallation directory path."
---

```ts
getPluginDirPath(): Promise<string | null | undefined>;
```

**Returns**

- `Promise<string | null | undefined>`:plugininstallation directory path

## Example

```ts wrap
import { PluginManager } from 'sn-plugin-lib';

/**
 * Get plugindirectorypath example.
 */
export async function exampleGetPluginDirPath() {
 const dir = await PluginManager.getPluginDirPath();
 return dir;
}
```
