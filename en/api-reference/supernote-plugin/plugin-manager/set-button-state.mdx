---
title: "setButtonState"
description: "Set buttonenable/disablestate."
---

```ts
setButtonState(id: number, state: boolean): Promise<boolean>;
```

**Parameters**

| Parameter | Type | Description |
| --- | --- | --- |
| `id` | `number` | buttonunique identifier(registerbuttonwhen Set id) |
| `state` | `boolean` | buttonstate:`true` enable,`false` disable |

**Returns**

- `Promise<boolean>`:Set result

## Example

```ts wrap
import { PluginManager } from 'sn-plugin-lib';

/**
 * Set buttonstate example.
 */
export async function exampleSetButtonState() {
 const ok = await PluginManager.setButtonState(1, false);
 return ok;
}
```
