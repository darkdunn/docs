---
title: "setButtonState"
description: "Set the state (enabled/disabled) of a button."
---

```ts
static setButtonState(id: number, enabled: boolean): Promise<boolean>;
```

**Parameters**

| Param | Type | Description |
| --- | --- | --- |
| `id` | `number` | Button ID |
| `enabled` | `boolean` | Enabled state |

**Returns**

- `Promise<boolean>`: `true` if updated successfully
