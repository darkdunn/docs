---
title: "getLayers"
description: "Get NOTE pagelayerdata."
---

```ts wrap
static getLayers(NOTEPath: string, page: number): Promise<APIResponse<Layer[]>>;
```

**Parameters**

| Parameter | Type | Description |
| --- | --- | --- |
| `NOTEPath` | `string` | NOTEfilepath |
| `page` | `number` | page number(starting from 0) |

**Returns**

- [`APIResponse<Layer[]>`](/en/api-reference/supernote-plugin/types/api-response):
`result` is pagelayerdataarray(type: see [Layer](/en/api-reference/supernote-plugin/types/layer))

## Example

```ts wrap
import { PluginFileAPI } from 'sn-plugin-lib';

/**
 * Get pagelayerdata example.
 */
export async function exampleGetLayers() {
 const NOTEPath = '/storage/emulated/0/Note/demo.note';
 const page = 1;
 const res = await PluginFileAPI.getLayers(NOTEPath, page);
 if (!res.success) {
 throw new Error(res.error?.message ?? 'getLayers call failed');
 }
 return res.result;
}
```
