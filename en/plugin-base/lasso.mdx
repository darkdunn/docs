## Overview

Lasso selects elements (strokes, text boxes, etc.) within a rectangular or arbitrary region. The SDK provides APIs to read lasso data, modify lasso geometries, and apply actions to selected elements.

## Common APIs

- Get lasso rectangle: [`getLassoRect`](/api-reference/supernote-plugin/plugin-comm-api/get-lasso-rect)
- Update lasso rectangle: [`updateLassoRect`](/api-reference/supernote-plugin/plugin-comm-api/update-lasso-rect)
- Get lasso elements: [`getLassoElements`](/api-reference/supernote-plugin/plugin-comm-api/get-lasso-data)
- Get element type counts: [`getLassoElementTypeCounts`](/api-reference/supernote-plugin/plugin-comm-api/get-lasso-data-type-counts)
- Modify lasso geometry: [`modifyLassoGeometry`](/api-reference/supernote-plugin/plugin-comm-api/modify-lasso-geometry)
