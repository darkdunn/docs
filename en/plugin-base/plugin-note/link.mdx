---
title: "Note Links"
description: "Working with text and stroke links in notes."
---

## Link Types

- Text link: style, font size, display text
- Stroke link: associate strokes via `controlTrailNums`

## APIs

- Get link: [`getLassoLink`](/api-reference/supernote-plugin/plugin-note-api/get-lasso-link)
- Set stroke link: [`setLassoStrokeLink`](/api-reference/supernote-plugin/plugin-note-api/set-lasso-stroke-link)
- Insert text link: [`insertTextLink`](/api-reference/supernote-plugin/plugin-note-api/insert-text-link)
- Modify link: [`modifyLassoLink`](/api-reference/supernote-plugin/plugin-note-api/modify-lasso-link)
