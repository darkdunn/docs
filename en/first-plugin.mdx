## Quickstart

This guide walks through creating a plugin project, registering buttons, and packaging your plugin for installation on Supernote devices.

### Create a Project

Use your preferred React Native setup to initialize a project.

### Register a Button

Integrate with PluginManager to register and handle button events. See [register-button](\/api-reference\/supernote-plugin\/plugin-manager\/register-button).

### Package and Install

Follow your build scripts to produce an APK and install it on a Supernote device.
