---
title: "Environment Setup"
description: "Set up React Native and Android build environments for Supernote plugin development."
---

## Requirements

- Node.js and Yarn
- Android SDK and platform tools
- React Native CLI or scripts provided by your project

## Steps

1. Install Node.js and Yarn
2. Install Android Studio and SDKs
3. Configure environment variables (JAVA_HOME, ANDROID_HOME, etc.)
4. Verify React Native build with a sample project
