## 快速定位（按你想做的功能）

### 文档（Document）能力
- 划词/选中文本：获取选中文本、获取文档页文本、获取文档总页数
- 适用场景：翻译/查词/摘录/总结/检索增强

### 笔记（Note）内容能力
- 链接：创建/修改套索链接、插入文字链接
- 标题：设置/获取/修改套索标题
- 文本框：获取/插入/修改文本框

### 文件与页面（File & Page）批处理能力
- 读写笔划数据：读取/插入/修改/替换页面 trails
- 页面结构：插入页面、删除页面
- 资源导出：生成页面 PNG、生成模板 PNG、生成标记缩略图
- 元数据：关键词、标题列表、页面模板、笔记类型/设备类型
- 图层：读取/修改/插入/删除/排序图层、清理图层数据

### 套索（Lasso）与几何能力
- 套索框：开关状态、获取/更新套索矩形
- 套索数据：统计类型数量、删除套索数据
- 几何图形：获取/插入/修改几何

### 贴纸（Sticker）工作流能力
- 从套索保存贴纸
- 贴纸尺寸/缩略图生成
- trail 转贴纸、插入贴纸到页面

### 宿主 UI 与系统控制
- 下拉菜单/滑条锁定
- 休眠状态控制
- 滑条显示状态控制

## 能力边界与注意事项

### 支持范围（能做 / 不能做）
- 能做：对笔记/文档内容做读取、插入、修改；响应按钮事件；生成资源文件
- 不能做（或需谨慎）：跨应用能力、直接调用底层 C/C++、未经宿主允许的系统级能力

### 返回值与错误处理
- APIResponse 结构说明（success/result/error）
- 常见参数校验失败原因（类型/必填/范围）

## 对应 API 模块（按类名查字典）

### PluginCommAPI（通用交互 / 套索 / 贴纸 / UI 控制）
- Trail 创建与回收
- 套索框与套索数据
- 几何图形
- 贴纸全链路
- UI/系统状态

### PluginNoteAPI（笔记语义：链接/标题/文本框）
- 套索链接与文字链接
- 套索标题
- 文本框

### PluginDocAPI（文档：选中文本与页文本）
- 选中文本
- 页文本与总页数

### PluginFileAPI（文件/页面级读写与批处理）
- 页面 trails 读写
- 页面增删与模板
- 导出与缩略图
- 关键词/标题/标记
- 图层

## 推荐从哪些能力开始做（新手路线）

### 入门（1 天内可做）
- 划词翻译/查词
- 套索转贴纸并插入
- 插入一个简单文本框

### 进阶（2–5 天）
- 读取 trails 做批量处理（替换/插入/修改）
- 目录结构化（标题/链接联动）
- 图层批量管理
